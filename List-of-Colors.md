# List of Common Colors (BGR Format)

- **Black**: `(0, 0, 0)`
- **White**: `(255, 255, 255)`
- **Red**: `(0, 0, 255)`
- **Green**: `(0, 255, 0)`
- **Blue**: `(255, 0, 0)`
- **Cyan**: `(255, 255, 0)`
- **Magenta**: `(255, 0, 255)`
- **Yellow**: `(0, 255, 255)`
- **Orange**: `(0, 165, 255)`
- **Pink**: `(203, 192, 255)`
- **Purple**: `(128, 0, 128)`
- **Gray**: `(128, 128, 128)`
- **Light Gray**: `(211, 211, 211)`
- **Dark Gray**: `(169, 169, 169)`
